# Employee Attrition Prediction

## Overview

This project predicts employee attrition using machine learning techniques.

The project was restructured from a Kaggle notebook into a modular Python project for better organization, readability, and maintainability.

---

## Dataset

IBM HR Analytics Employee Attrition Dataset

---

## Project Structure

employee_attrition_prediction/

├── data/
├── preprocessing/
├── models/
├── evaluation/
├── utils/
├── tests/
├── main.py
├── requirements.txt
└── README.md

---

## Models Used

- Logistic Regression
- Random Forest Classifier

---

## Features

- Data preprocessing
- Feature encoding
- Train-test split
- Model training
- Model evaluation
- Confusion matrix visualization
- Unit testing
